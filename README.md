## Overview

**UNO Game** is a console-based implementation of the classic UNO card game developed in **C++** as a Programming Fundamentals course project.

The game supports two-player gameplay and implements the core mechanics of UNO, including card shuffling, special action cards, score tracking, and file handling using a procedural programming approach.

---

## Features

- 2-player turn-based gameplay
- Randomized deck shuffling and card dealing
- Colored console interface
- Special cards
  - Skip
  - Reverse
  - Draw Two
  - Wild
  - Wild Draw Four
- UNO call functionality
- Persistent score saving
- Interactive menu system

## Getting Started

### Prerequisites

- Windows Operating System
- Embarcadero Dev-C++

### Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/UNO-Game.git
```

Open **UNO Game.cpp** in **Embarcadero Dev-C++**, compile, and run the project.

---

## Controls

| Key | Action |
|-----|--------|
| **H** | Play a card |
| **Q** | Draw a card |
| **U** | Call UNO |
| **S** | Save Game |
| **E** | Exit / Return to Main Menu |
